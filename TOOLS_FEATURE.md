## 图片尺寸统一工具

### 功能特性

1. **批量上传**
   - 支持点击上传或拖拽上传
   - 支持 JPG、PNG、WebP、GIF 格式
   - 最多支持 5000 张图片

2. **四种调整模式**
   - **等比缩放**：保持原图比例，适应目标尺寸
   - **填充**：填充目标区域，可能产生留白
   - **裁剪**：居中裁剪到目标尺寸
   - **智能自动**：根据设定的尺寸范围自动处理
     - 最长边 < 最小边 → 放大到最小边
     - 最长边 > 最大边 → 缩小到最大边
     - 最长边在范围内 → 保持原样

3. **输出设置**
   - 目标宽度/高度（像素）
   - 保持宽高比选项
   - 输出格式：JPEG / PNG / WebP
   - 图片质量：1-100

4. **预览功能**
   - 原图预览：显示文件名、原始尺寸、文件大小
   - 效果预览：显示处理后的效果和预估大小

5. **批量处理**
   - 实时进度显示
   - 处理完成后显示统计信息（原始大小、处理后大小、节省空间）

6. **下载选项**
   - 单独下载所有图片
   - 打包下载为 ZIP 文件

### 设置持久化

用户的设置会自动保存到 localStorage，下次打开时自动恢复。

## UI 设计

- 深色主题，与主应用界面统一
- 使用 brand-500 作为主色调
- 圆角卡片设计
- 响应式布局

## 文件结构

```
components/
├── ImageResizer.tsx  # 图片处理核心组件
└── ToolsModal.tsx    # 工具箱模态框容器
```

## 扩展性

ToolsModal 组件设计为可扩展的工具容器，未来可以轻松添加更多工具，只需：

1. 创建新的工具组件
2. 在 `ToolsModal.tsx` 的 `tools` 数组中添加工具配置
3. 在内容区域添加对应的渲染逻辑

## 依赖

- `jszip`: 用于打包下载 ZIP 功能
