# RunningHub AI 客户端 v1.1.0 更新公告

**发布日期**: 2026年1月16日

---

## ✨ 新增功能

### 1. 应用库管理
- 可以导入并保存常用的 AI 应用
- 支持自定义应用名称和颜色
- 应用列表永久保存，无需重复配置

### 2. 后台任务系统
- 支持同时运行最多 3 个任务
- 超过限制自动排队，无需手动等待
- 实时显示任务进度和状态

### 3. 可拖拽任务管理器
- 右下角浮窗显示所有任务状态
- 可以拖动到屏幕任意位置
- 位置自动保存，下次打开还在原位
- 任务完成/失败会弹出通知

### 4. 任务控制
- 可以随时取消正在运行的任务
- 可以删除单个结果图片
- 可以一键清空已完成的历史记录

### 5. 历史记录升级
- 从九宫格图片升级为详细卡片视图
- 显示任务 ID、运行时长、完成时间等信息
- 支持展开/收起查看详细信息
- 图片采用圆角设计，视觉更美观

### 6. 全局设置
- 可以保存 RunningHub API Key 和 Access Token
- 设置永久记住，无需每次输入

1. 统一圆角卡片设计风格

4. 完善错误提示和异常处理

## 💡 使用提示

- **首次使用**：先在左侧应用库导入你的 AI 应用
- **快速访问**：任务管理器可以拖到顺手的位置
- **查看详情**：点击历史记录右上角展开按钮查看任务详情
- **快捷操作**：鼠标悬浮在图片上可以快速预览和下载
