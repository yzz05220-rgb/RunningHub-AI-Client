# RunningHub AI 客户端 - 功能更新说明

> 一份简单易懂的版本更新说明，让你快速了解新增了哪些好用的功能！

---

## 🎉 我们新增了什么功能？

### 1. **应用管理** - 像管理手机App一样管理AI应用

**以前怎么做**：每次用AI都要输入一大堆配置  
**现在怎么做**：
- ✨ 把常用的AI应用添加到"应用库"
- ✨ 给每个应用取个好记的名字，选个喜欢的颜色
- ✨ 一键点击就能使用，再也不用重复输入

**怎么用**：
1. 点击左侧的"应用库"
2. 输入 AI 应用的 ID
3. 起个名字，选个颜色
4. 确定！以后直接点击就能用

---

### 2. **后台任务** - 可以同时运行多个任务啦

**以前怎么做**：运行一个任务，就得等它完成才能继续  
**现在怎么做**：
- ✨ 最多同时运行 3 个任务
- ✨ 超过 3 个？自动排队，不用手动等
- ✨ 可以随时取消不想要的任务

**你能看到**：
- 每个任务的进度条（知道还要等多久）
- 任务是在运行、排队还是已完成
- 任务成功或失败的通知提示

---

### 3. **任务管理器** - 随时查看所有任务状态

**长什么样**：右下角有个浮窗，显示所有任务的状态

**能做什么**：
- ✨ 看到正在运行的任务实时进度
- ✨ 点击展开查看详细列表
- ✨ 随便拖动到屏幕任意位置
- ✨ 位置会被记住，下次打开还在那

**超级方便**：
- 点击就能预览结果图片
- 不喜欢的可以直接删除
- 失败的任务会显示失败原因

---

### 4. **历史记录增强** - 找回以前生成的图片更方便

**以前怎么样**：只能看到小图片，什么信息都没有  
**现在怎么样**：
- ✨ 每个任务都有详细卡片
- ✨ 显示是哪个应用生成的
- ✨ 显示什么时候完成的
- ✨ 显示运行了多长时间
- ✨ 失败的会显示错误原因

**更好的浏览体验**：
- 图片变大了，看得更清楚
- 点击可以全屏查看
- 一键下载保存
- 可以点右上角展开查看更多细节

---

### 5. **设置中心** - 保存你的API密钥

**在哪里**：右上角齿轮图标  
**能设置什么**：
- RunningHub API Key（API密钥）
- Access Token（访问令牌）

**好处**：设置一次，永久记住，不用每次都输

## ❓ 常见问题

**Q: 历史记录会保存多久？**  
A: 永久保存在你的电脑上，除非你手动清空。

**Q: 可以保存多少个应用？**  
A: 无限制，想加多少加多少。

**Q: 任务管理器可以关闭吗？**  
A: 当前不能关闭，但我们计划加入最小化功能。

**Q: 同时运行的任务数可以调整吗？**  
A: 当前固定是3个，未来会加入自定义选项。

**Q: 支持手机使用吗？**  
A: 目前是桌面版，暂不支持手机。

## 💡 使用小技巧

1. **快速开始**：先在应用库添加常用的AI应用
2. **批量处理**：等批量功能上线后，一次处理多张图更高效
3. **定期清理**：定期清空不需要的历史记录，保持整洁
4. **拖动定位**：把任务管理器拖到顺手的位置
5. **展开查看**：点击历史记录的展开按钮查看详细信息
